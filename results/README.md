# Results Artifacts

This directory stores small submission artifacts only.

Expected contents:

- `logs/` — text or JSONL logs for baseline inference, improved inference, TFDS build, and short LoRA tests.
- `screenshots/` — screenshots for dataset/build/inference evidence.
- `episode_visualizations/` — generated episode preview grids.

Large generated datasets, TFRecords, model checkpoints, and safetensors must not be committed.
